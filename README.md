# 🌿 Plant Doctor AI 3.0

An AI-powered plant disease detection application built using TensorFlow, EfficientNetB0, and Streamlit.

## Features

- 🌱 Detect plant diseases from leaf images
- 🤖 EfficientNetB0 deep learning model
- 📊 Top 3 predictions with confidence scores
- 💊 Disease information and treatment suggestions
- 📜 Scan history
- 🌐 Streamlit web interface

## Technologies Used

- Python
- TensorFlow
- Keras
- Streamlit
- Pillow
- NumPy

## Dataset

PlantVillage Dataset
